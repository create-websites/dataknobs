Data Centric AI | Using Knobs

Data Knobs is Data Centric AI company. Our mission is make data useble and useful. Instead of focusing on sophisticated algorithms we help businesses in buiding gold datasets. We use variety of approaches to build datasets for AI. We create new data, augment dataset, anonymize dataset. From Big data we focus on abstracting state action space so that optimal policy and learning can be done with well governed datasets.
