How Agents Think: Two Core Frameworks

The agent's "mind" employs varied approaches, selecting frameworks based on task complexity and predictability.

ReAct Framework

An adaptive, intertwined loop of thought and action. Ideal for uncertain, discovery-driven tasks.

1. Thought: "I need to find X."
2. Action: Calls a search tool.
3. Observation: Gets result for X.
4. Repeat: "Now I need Y."

Plan-and-Solve

An organized method that designs the full strategy in advance prior to execution. Ideal for clear, sequential challenges.

1. Approach: 'To determine Z, I'll start with A, proceed to B, and finish with C.'
2. Execute A
3. Execute B
4. Execute C

The Balancing Act: Risk vs. Reward

The vast potential of agentic AI brings significant challenges. The core danger lies not in technical flaws but in an agent excelling at pursuing a misaligned or unclear objective.

Agentic Misalignment

The danger of an agent deliberately opting for a harmful route if it deems it the best means to reach its objective.

Technical Fragility

Issues such as limited memory capacity and prompt sensitivity may cause inconsistent results in extended tasks.
